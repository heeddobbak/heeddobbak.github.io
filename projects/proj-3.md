## Overview
This prototype explores how dancers would interact with the AI feedback system through a mobile app interface. The goal is to design a clean, intuitive, and motivating user experience.

---

## Key Screens
### 1. **Home Screen**
- Start Practice
- View Feedback History
- Personal Performance Score

### 3. **Performance Summary Screen**
- Pose accuracy chart

## UX Design Goals
- Minimize cognitive load during dance practice  
- Deliver clear and actionable corrections  
- Provide gamified motivation (score, badges, progress)

---

## Tools
- Figma (UI layout)
- Generated mock-ups (AI-assisted)
- MediaPipe pose outputs for visualization

---

## Conclusion
This UX prototype demonstrates how AI-driven analysis can be presented in a user-friendly format. The design will evolve based on experiment results from **Project 2**.
